# Platform Differences

sunnypilot runs on different hardware platforms, each with its own settings interface. Not every setting is available on every device. This page explains the differences so you know what to expect.

## Supported Platforms

### comma 3X / comma 3 (TIZI/TICI)

The full-featured interface with a sidebar-based settings panel. All settings documented in this manual are available on these devices, giving you complete control over steering, cruise, visuals, display, models, maps, and vehicle-specific configuration.

### comma four (MICI)

A simplified, touch-friendly interface designed for the comma four's compact form factor. Only essential settings are exposed through large buttons and toggles. Many advanced driving controls (steering tuning, cruise customization, visual overlays, model selection) are not available through this interface.

### sunnylink (Web / App)

The sunnylink web interface shows all available settings regardless of which device you use. If a setting is missing from your comma four's on-device screen, you can still configure it through sunnylink.

## What This Means for You

If you use a **comma four**, your on-device settings screen shows a simplified set of controls focused on essential device management and basic feature toggles. For the full range of settings, use **sunnylink** from your browser.

If you use a **comma 3X or comma 3**, all settings are directly accessible from the device sidebar.

!!! tip
    Throughout this manual, each setting includes a "Supported Devices" note indicating which platforms can access it. Look for these callouts to quickly check if a setting is available on your device.
